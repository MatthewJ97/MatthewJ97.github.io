Cross-Platform Scissors-Paper-Rock

This Project was created for a complex systems assessment, It uses Unity's UNET system to display the server addresses of available servers on a LAN allowing clients to connect to the server. within this i created an app for the iphone that played scissors paper, rock utlising this system.

Tasks

Programming Contributions:

  • Account Class: This class was the basis of the player where it held the username, password and player i, to create the player id i added the player username and password to one string then adding each of those ascii characters numeric values together and adding 10000.

  • Network Controller: A simple network controller that housed unitys UNET functions wrapped within my own

  • Override Network Discovery: Allowed for devices to find the host ip

  • Player: A more complex script that controlled the players interactions with the server and GUI, as well as handling the players account and gameplay information.

  • Server: controlled the time and gameplay between the players. It also controlled the host GUI.
