Tilemap Creator/Editor

This tile map creator was made for another Academy assessment. it was created using WPF. The aim of this project was to make a tool that could export files for external tools, to achieve this i allowed for the project to export the tiles to symbols in a text file that could be read into another project.

Tasks

Programming Contributions:

  • Tile Class: Each tile in the editor has its own settings that are controlled by this class, this class also controls the gui and mouse controls within the editor for each tile.

  • NewMapWindow: Controls the creation of a new map, closing a window.

  • MainWindow controls the main portion of the program. this includes loading in images for the maps, generating maps, window resizing, and exiting.
