OpenGL

This project utilises the OpenGL Library, the scene uses custom shaders for rendering of objects within the scene which have been loaded through an FBX loader library. These custom shaders calculate effects trhough post rendering. Heightmap calculations are made from a texture to create a "terrain".One of the models within the scene uses custom frustum culling calculations for educational purposes. Spritesheets have been used to create animations for bullets.
