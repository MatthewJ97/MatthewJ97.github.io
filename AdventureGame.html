Adventure Game

Simple Text adventure game created in console. This Project was created as part of my first assignment at AIE. I improved aupon my original assignment slight to make it a little less buggy and use better structure. It could be improved further for both design and functionality. As part of this assignment we had to create our own string container to work in the console.

Tasks

Programming Contributions:

  • String Class: This class was created to allow for the basic string functionality, this included constructors, a Destructor, Length, CharacterAt, EqualTo, Append, Prepend, Cstr, toLower, ToUpper, find and find at Index, replace and set

  • Player: A simple player class that held all the players stats

  • Enemy: A simple class that handles the spawning of enemys of different types, Attacking and hold the stats. This class was going to be used to inherit the enemy from but due to the early stages of my programming skills i didnt have the knowledge.

  • Room: A simple class that handles the navigation through the rooms of the game by checking the players input against those of the current room. BACK
